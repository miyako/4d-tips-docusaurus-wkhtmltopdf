Lista de propiedades JSON

Esta página ofrece una lista completa de todas las propiedades de los formularios, ordenadas por su nombre JSON. Haga clic en el nombre de una propiedad para acceder a su descripción detallada.

En el capítulo "Propiedades del formulario", las propiedades se ordenan según sus nombres y temas en la lista de propiedades.

a - c - d - e - f - h - i - m - p - r - s - w

Propiedad Descripción Valores posibles
a
bottomMargin Valor del margen vertical (en píxeles) mínimo: 0
c
colorScheme Esquema de colores para el formulario "dark", "light"
d
destination Tipo de formulario "detailScreen", "listScreen", "detailPrinter", "listPrinter"
e
entryOrder El orden en el cual los objetos activos son seleccionados cuando la tecla Tab o la tecla Retorno de carro se utilizan en un formulario de entrada Colección de nombres de objetos 4D Form
events Lista de todos los eventos seleccionados para el objeto o el formulario Colección de nombres de eventos, por ejemplo ["onClick", "onDataChange"...].
f
formSizeAnchor Nombre del objeto cuya posición determina el tamaño del formulario. (longitud mínima: 1) Nombre de un objeto 4D
h
height Altura del formulario mínimo: 0
i
inheritedForm Designa el formulario a heredar Nombre (cadena) de la tabla o formulario proyecto O una ruta POSIX (cadena) a un archivo .json que describa el formulario O un objeto que describa el formulario
inheritedFormTable Designa la tabla que utilizará un formulario heredado Un nombre o número de tabla
m
markerBody Posición del marcador de detalle mínimo: 0
markerBreak Posición del marcador de ruptura mínimo: 0
markerFooter Posición del marcador de pie mínimo: 0
markerHeader Posición del marcador de encabezado integer minimum: 0; integer array minimum: 0
memorizeGeometry Guarda los parámetros del formulario cuando se cierra la ventana del mismo true, false
menuBar Barra de menú a asociar al formulario Nombre de una barra de menú válida
method Un nombre de método proyecto. El nombre de un método proyecto existente
p
pages Colección de páginas (cada página es un objeto) Objetos página
pageFormat objeto Propiedades de impresión disponibles
r
rightMargin Valor del margen horizontal (en píxeles) mínimo: 0
s
shared Especifica si un formulario puede utilizarse como subformulario true, false
w
ancho Ancho del formulario mínimo: 0
windowMaxHeight La mayor altura permitida de la ventana del formulario mínimo: 0
windowMaxWidth El mayor ancho permitido de la ventana de formulario mínimo: 0
windowMinHeight La menor altura permitida de la ventana de formulario mínimo: 0
windowMinWidth El menor ancho permitido de la ventana de formulario mínimo: 0
windowSizingX Dimensionamiento vertical de la ventana del formulario "fixed", "variable"
windowSizingY Dimensionamiento horizontal de la ventana del formulario "fixed", "variable"
windowTitle Designa el título de una ventana de formulario Un nombre para la ventana de formulario
