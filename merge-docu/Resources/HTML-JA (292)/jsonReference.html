JSON プロパティリスト

この一覧は、すべてのフォームプロパティを JSON名順に並べてまとめたものです。 プロパティ名をクリックすると、詳細説明に移動します。

"フォームプロパティ" の章では、プロパティリストにおけるテーマと名称で各プロパティを紹介しています。

a - c - d - e - f - h - i - m - p - r - s - w

プロパティ 説明 とりうる値
a
bottomMargin 垂直マージン値 (ピクセル単位) 最小値: 0
c
colorScheme フォームのカラースキーム "dark", "light"
d
destination フォームタイプ "detailScreen", "listScreen", "detailPrinter", "listPrinter"
e
entryOrder 入力フォームで Tabキーや 改行キーが使用されたときに、アクティブオブジェクトが選択される順番。 4Dフォームオブジェクト名のコレクション
events オブジェクトまたはフォームについて選択されているイベントのリスト イベント名のコレクション。例: ["onClick","onDataChange"...].
f
formSizeAnchor フォームサイズを定義するために使用するオブジェクトの名前 (最小長さ: 1) 4Dオブジェクトの名前
h
height フォームの高さ 最小値: 0
i
inheritedForm 継承フォームを指定します テーブルまたはプロジェクトフォームの名前 (文字列), フォームを定義する .json ファイルへの POSIX パス (文字列), またはフォームを定義するオブジェクト
inheritedFormTable 継承フォームがテーブルに属していれば、そのテーブル テーブル名またはテーブル番号
m
markerBody 詳細マーカーの位置 最小値: 0
markerBreak ブレークマーカーの位置 最小値: 0
markerFooter フッターマーカーの位置 最小値: 0
markerHeader ヘッダーマーカーの位置 最小値 (整数): 0; 最小値 (整数配列): 0
memorizeGeometry フォームウィンドウが閉じられた時に、フォームパラメーターを記憶 true, false
menuBar フォームと関連づけるメニューバー 有効なメニューバーの名称
method プロジェクトメソッド名 存在するプロジェクトメソッドの名前
p
pages ページのコレクション (各ページはオブジェクトです) ページオブジェクト
pageFormat object 利用可能な印刷プロパティ
r
rightMargin 水平マージン値 (ピクセル単位) 最小値: 0
s
shared フォームがサブフォームとして利用可能かを指定します true, false
w
width フォームの幅 最小値: 0
windowMaxHeight フォームウィンドウの最大高さ 最小値: 0
windowMaxWidth フォームウィンドウの最大幅 最小値: 0
windowMinHeight フォームウィンドウの最小高さ 最小値: 0
windowMinWidth フォームウィンドウの最小幅 最小値: 0
windowSizingX フォームウィンドウの高さ固定 "fixed", "variable"
windowSizingY フォームウィンドウの幅固定 "fixed", "variable"
windowTitle フォームウィンドウのタイトルを指定します フォームウィンドウの名前。
